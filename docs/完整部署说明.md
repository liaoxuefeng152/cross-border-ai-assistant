## 📞 故障排除

### 问题1：前端无法连接后端
**解决方案：**
1. 检查后端是否运行：`curl https://longzhangguai-backend.loca.lt/health`
2. 如果后端停止，重启服务：
   ```bash
   cd /workspace/projects/backend
   /usr/bin/python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000
   ```

### 问题2：内网穿透断开
**解决方案：**
```bash
# 重启前端穿透
npx localtunnel --port 8080 --subdomain longzhangguai-ai

# 重启后端穿透
npx localtunnel --port 8000 --subdomain longzhangguai-backend
```

### 问题3：API响应慢
**原因：** 内网穿透有延迟
**解决方案：** 这是正常的，免费版有速度限制
