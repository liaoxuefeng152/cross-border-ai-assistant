## 🔧 服务管理

### 检查服务状态
```bash
# 检查HTTP服务器
ps aux | grep "python3 -m http.server 8080"

# 检查内网穿透
ps aux | grep "localtunnel"
```

### 重启服务
```bash
# 停止服务
pkill -f "python3 -m http.server 8080"
pkill -f "localtunnel"

# 重启HTTP服务器
cd /workspace/projects/assets
nohup python3 -m http.server 8080 > /tmp/http_server.log 2>&1 &

# 重启内网穿透
nohup npx localtunnel --port 8080 --subdomain longzhangguai-ai > /tmp/lt.log 2>&1 &
```

### 查看日志
```bash
# 查看HTTP服务器日志
tail -f /tmp/http_server.log

# 查看内网穿透日志
tail -f /tmp/lt.log
```
