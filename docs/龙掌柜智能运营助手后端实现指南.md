# 龙掌柜智能运营助手 - 后端系统完整实现指南

## 项目概述

请为龙掌柜智能运营助手实现完整的后端AI运营系统。目前前端已有聊天界面，需要实现完整的后端功能，包括智能对话、商品采集、Listing生成、定价分析等全流程运营能力。

## 三、核心功能模块

### 3.1 智能对话系统
**功能描述**: 实现基于LangGraph的智能对话Agent，支持多轮对话和上下文理解

**核心要求**:
- 支持多轮对话，保留最近20轮会话记录
- 实现用户意图识别
- 支持工具调用
- 会话状态管理

**实现要点**:
- 使用LangGraph构建Agent
- 实现会话记忆（MemorySaver）
- 支持异步调用
- 错误处理和降级

### 3.2 商品信息采集
**功能描述**: 支持从1688、teum、淘宝、天猫、京东、拼多多等平台采集商品信息

**支持平台**:
- 1688 (阿里巴巴批发)
- teum (跨境电商)
- 淘宝 (C2C)
- 天猫 (B2C)
- 京东 (综合电商)
- 拼多多 (社交电商)

**采集字段**:
- 商品标题
- 商品价格
- 商品图片
- 商品描述
- 商品规格
- 销量数据
- 评价数据

**实现要点**:
- 自动识别平台
- 提取商品ID
- 解析商品页面
- 数据清洗和格式化
- 错误处理和重试

### 3.3 图片AI处理
**功能描述**: 使用AI生成商品主图、生活场景图、信息图

**生成类型**:
- 商品主图（白底图）
- 生活场景图
- 信息图（卖点图）
- 详情页图片

**输出规格**:
- Shopee: 800x800 (主图), 1024x1024 (详情图)
- Lazada: 800x800 (主图), 1024x1024 (详情图)
- TikTok Shop: 800x800 (主图), 1024x1024 (详情图)
- Amazon: 1000x1000 (主图), 1000x1000 (详情图)

### 3.4 AI视频生成
**功能描述**: 生成商品展示视频和介绍视频

**视频类型**:
- 商品展示视频（15-60秒）
- 带解说的介绍视频（30-90秒）
- 使用场景视频

**视频规格**:
- TikTok: 720p/1080p, 15-60秒
- Shopee: 480p/720p, 10-30秒
- Lazada: 480p/720p, 10-30秒

### 3.5 Listing智能生成
**功能描述**: 根据商品信息生成高质量的Listing内容

**生成内容**:
- 优化标题（包含关键词）
- 五点描述（突出卖点）
- 详细描述（专业文案）
- 搜索关键词
- SEO优化

**多语言支持**:
- 英文（通用）
- 印尼语（印尼站）
- 泰语（泰国站）
- 越南语（越南站）
- 马来语（马来西亚站）
- 菲律宾语（菲律宾站）

### 3.6 智能定价
**功能描述**: 基于成本、市场、竞争等因素智能定价

**定价因素**:
- 供货成本
- 运费成本
- 平台佣金
- 广告费用
- 目标利润率
- 市场价格
- 竞争程度

**输出结果**:
- 保本价
- 保守定价
- 平衡定价
- 激进定价
- 价格战预警

### 3.7 利润分析
**功能描述**: 实时分析销售数据和利润情况

**分析维度**:
- 每日利润
- 每周利润
- 单品利润
- 类目利润
- 成本拆分（供货、运费、佣金、广告）

**输出内容**:
- 利润报表
- 亏损商品识别
- 优化建议

### 3.8 智能客服
**功能描述**: 自动翻译和回复客户咨询

**支持语言**:
- 英文
- 印尼语
- 泰语
- 越南语

- 菲律宾语

**回复类型**:

### 3.9 合规检查
**功能描述**: 检查Listing和图片的合规性

**检查项目**:

**输出结果**:
- 健康评分

### 3.10 商品发布
**功能描述**: 生成符合各平台规范的商品数据包

**支持平台**:

**输出内容**:

## 四、数据库设计

### 4.1 用户表 (users)
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    hashed_password VARCHAR(255) NOT NULL,
    full_name VARCHAR(100),
    is_active BOOLEAN DEFAULT TRUE,
    is_superuser BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4.2 会话表 (conversations)
```sql
CREATE TABLE conversations (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    session_id VARCHAR(100) UNIQUE NOT NULL,
    title VARCHAR(200),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4.3 消息表 (messages)
```sql
CREATE TABLE messages (
    id SERIAL PRIMARY KEY,
    conversation_id INTEGER REFERENCES conversations(id),
    role VARCHAR(20) NOT NULL, -- 'user' or 'assistant'
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4.4 商品表 (products)
```sql
CREATE TABLE products (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    source_platform VARCHAR(50), -- 1688, taobao, etc.
    source_product_id VARCHAR(100),
    title VARCHAR(500),
    price DECIMAL(10, 2),
    images JSONB,
    description TEXT,
    attributes JSONB,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 4.5 发布记录表 (published_products)
```sql
CREATE TABLE published_products (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    product_id INTEGER REFERENCES products(id),
    target_platform VARCHAR(50), -- Shopee, Lazada, etc.
    target_product_id VARCHAR(100),
    status VARCHAR(20), -- 'pending', 'published', 'failed'
    published_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

```python
from fastapi import APIRouter, HTTPException, Depends, BackgroundTasks
from sqlalchemy.ext.asyncio import AsyncSession
from typing import Optional
from app.api.deps import get_current_user, get_db
from app.schemas.chat import ChatRequest, ChatResponse, ChatHistoryResponse
from app.agents.agent import build_agent
from app.models.user import User
from langchain_core.messages import HumanMessage, AIMessage

router = APIRouter(prefix="/chat", tags=["chat"])

@router.post("/message", response_model=ChatResponse)
async def chat_message(
    request: ChatRequest,
    current_user: User = Depends(get_current_user),
    db: AsyncSession = Depends(get_db)
):
    """
    发送消息并获取AI回复

    Args:
        request: 聊天请求，包含消息和会话ID
        current_user: 当前用户
        db: 数据库会话

    Returns:
        ChatResponse: AI回复和会话信息
    """
    try:
        # 构建Agent
        agent = build_agent()

        # 调用Agent
        result = await agent.ainvoke(
            {"messages": [HumanMessage(content=request.message)]},
            config={"configurable": {"thread_id": request.session_id}}
        )

        # 提取回复
        messages = result.get("messages", [])
        ai_messages = [msg for msg in messages if isinstance(msg, AIMessage)]

        if not ai_messages:
            raise HTTPException(status_code=500, detail="AI未生成回复")

        reply = ai_messages[-1].content

        # 保存会话记录（异步）
        await save_conversation(
            db=db,
            user_id=current_user.id,
            session_id=request.session_id,
            user_message=request.message,
            assistant_message=reply
        )

        # 提取使用的工具
        tools_used = []
        for msg in messages:
            if hasattr(msg, 'tool_calls') and msg.tool_calls:
                for tool_call in msg.tool_calls:
                    tools_used.append(tool_call.get('name', ''))

        return ChatResponse(
            message=reply,
            session_id=request.session_id,
            tools_used=list(set(tools_used))
        )

    except Exception as e:
        import traceback
        traceback.print_exc()
        raise HTTPException(status_code=500, detail=f"处理失败: {str(e)}")

@router.get("/history", response_model=ChatHistoryResponse)

async def save_conversation(db, user_id, session_id, user_message, assistant_message):
    """保存对话记录"""
    from app.models.conversation import Conversation, Message
    from datetime import datetime

    # 查找或创建会话
    conv = await db.execute(
        select(Conversation).where(
            Conversation.user_id == user_id,
            Conversation.session_id == session_id
        )
    )
    conversation = conv.scalar_one_or_none()

    if not conversation:
        conversation = Conversation(
            user_id=user_id,
            session_id=session_id,
            title=user_message[:50] if user_message else "新对话"
        )
        db.add(conversation)
        await db.flush()

    # 保存用户消息
    user_msg = Message(
        conversation_id=conversation.id,
        role="user",
        content=user_message
    )
    db.add(user_msg)

    # 保存助手消息
    assistant_msg = Message(
        conversation_id=conversation.id,
        role="assistant",
        content=assistant_message
    )
    db.add(assistant_msg)

    conversation.updated_at = datetime.now()
    await db.commit()

async def get_conversation_history(db, user_id, session_id):
    """获取对话历史"""
    from app.models.conversation import Message
    from sqlalchemy import select

    result = await db.execute(
        select(Message)
        .join(Conversation)
        .where(
            Conversation.user_id == user_id,
            Conversation.session_id == session_id
        )
        .order_by(Message.created_at)
    )
    messages = result.scalars().all()

    return [
        {"role": msg.role, "content": msg.content, "created_at": msg.created_at}
        for msg in messages
    ]
```

```python
def detect_platform(url: str) -> str:

def extract_product_id(url: str, platform: str) -> Optional[str]:

                    "status": "failed",
                    "error": str(e)
                })

        return json.dumps({
            "success": success_count,
            "failed": failed_count,
            "total": len(urls),
            "results": results
        }, indent=2, ensure_ascii=False)
```
