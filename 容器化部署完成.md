## 🚀 快速启动

### 方式1：使用部署脚本（最简单）
```bash
cd /workspace/projects
./deploy.sh
# 选择选项 1）构建并启动服务
```

### 方式2：使用 Make 命令
```bash
make up
```

### 方式3：使用 Docker Compose
```bash
docker-compose up -d
```

## 🔄 部署流程

```
1. 配置环境变量 (.env)
   ↓
2. 构建镜像 (docker-compose build)
   ↓
3. 启动服务 (docker-compose up -d)
   ↓
4. 健康检查 (test-docker.sh)
   ↓
5. 访问应用 (http://localhost:8080)
```

## 🔧 常用命令速查

```bash
# 启动
make up              # 启动服务
make restart         # 重启服务
make down            # 停止服务

# 日志
make logs            # 所有日志
make logs-backend    # 后端日志
make logs-frontend   # 前端日志

# 检查
make ps              # 服务状态
make check           # 健康检查
make stats           # 资源使用

# 构建
make build           # 构建镜像
make rebuild         # 重新构建
make clean           # 清理资源

# 导出
make export-all      # 导出所有镜像
```
